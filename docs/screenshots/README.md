## 권장 데모 프로필 (결과가 풍부하게 나옴)

- 사업자 유형: **법인**
- 설립연도: **2021** (업력 5년)
- 소재지: **서울**
- 수출 경험: **있음**
- 관심 분야: **해외수출 · 전시박람회**

→ 자격충족 53/65, 상단에 컨셉코리아·Texworld Paris·서울패션위크가 97~92점.
첫 카드(컨셉코리아)를 펼쳐 "왜 적합한가요" 블록이 보이게 캡처하면 차별점이 잘 드러납니다.

## 팁

- 너비 1200px 안팎으로 캡처하면 카드 밀도가 보기 좋습니다.
- macOS: `Cmd+Shift+4` 영역 캡처 또는 브라우저 개발자도구의 "노드 스크린샷".
